# Projects

Store one project per subdirectory, for example:

- project-abcde123/
  - PROJECT.md
  - data/

The project metadata should be declared in PROJECT.md frontmatter:

---
id: project-abcde123
name: Example project
description: Short summary
status: active
data_dir: data
tags: [demo, draft]
artifact_profile:
    skills: []
    scripts: []
    flows: []
    cases: []
---

Project details go below.
